## 🚀 GitHub Pages 배포 방법

### 방법 1: GitHub 웹사이트에서 배포 (가장 쉬움)

1. **GitHub 로그인** → https://github.com
2. **New Repository 클릭** (우측 상단 + 버튼)
3. **Repository 설정:**
   - Repository name: `sono-happy450-guide` (또는 원하는 이름)
   - Public 선택
   - "Add a README file" 체크 해제
   - Create repository 클릭

4. **파일 업로드:**
   - "uploading an existing file" 링크 클릭
   - `index.html` 파일을 드래그 앤 드롭
   - Commit changes 클릭

5. **GitHub Pages 활성화:**
   - Repository 페이지에서 **Settings** 클릭
   - 왼쪽 메뉴에서 **Pages** 클릭
   - Source: **Deploy from a branch** 선택
   - Branch: **main** 선택, 폴더: **/ (root)** 선택
   - Save 클릭

6. **배포 완료 확인:**
   - 1-2분 후 페이지 새로고침
   - "Your site is live at https://username.github.io/sono-happy450-guide/" 메시지 확인
   - 링크 클릭하여 사이트 확인!

### 방법 2: Git 명령어 사용 (개발자용)

```bash
# 1. Repository 생성 (GitHub 웹사이트에서)

# 2. 로컬에서 Git 초기화
git init
git add index.html README.md
git commit -m "Initial commit: Sono Happy450 Marketing Guide"

# 3. GitHub Repository 연결
git remote add origin https://github.com/your-username/your-repo-name.git
git branch -M main
git push -u origin main

# 4. GitHub Pages는 Settings에서 활성화 (방법 1의 5번 참고)
```

## 🔗 배포 후 URL

